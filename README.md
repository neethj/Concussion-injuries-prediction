# Concussion Injuries Prediction

This repository is a small R-based data analysis project focused on NFL concussion injury data from 2012–2014.

## What this project investigates

The core research question is whether offensive and defensive positions differ in injury recovery time (represented by `Week.of.Injury` in the dataset). The hypothesis metadata is documented in `research_question.yml`.

## Main files

- `Concussion Injuries 2012-2014.csv`: Source dataset used for analysis.
- `analysis.R`: Cleans and groups position data into offense/defense and runs a Wilcoxon test comparing `Week.of.Injury` between those groups.
- `visualization.R`: Produces a PDF with visualizations (position vs. week-of-injury box/strip chart and a week-of-injury histogram with an overlaid normal curve).
- `visualization.pdf`: Output chart artifact generated by the visualization script.
- `research_question.yml`: Project context (group, variables, hypotheses, and dataset provenance).

## How to run

1. Ensure R is installed.
2. From the repository root, run:

```r
source("analysis.R")
source("visualization.R")
```

The statistical test output prints to console, and charts are written to `visualization.pdf`.
